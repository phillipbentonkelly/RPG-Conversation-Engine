Role Playing Game (RPG) Conversation Engine

This project aims to provide a Conversation System for JavaScript-based games. The system aims to provide the functionality comparable to those seen in Role Playing Games (RPGs). The default implementation allow Turn-Based, Synchronous, Asynchronous character dialog and actions to be defined and performed. I will be building this system in phases, starting with the basics functions, turn-based conversation with no-actions. As I progress the other features will be added; WARNING: There will be a significant amound of re-writing of the engine as I add new features; Some of which will require dropping or changing previous work. Within this repo there will be a folder with previous (working) versions you can still use.

How to Use.

This project has a dependancy on jQuery, in this demo I am using 1.8.2, so include it with your resources.

  1. Import a jQuery library into your document. 
    jquery.1.8.2.min.js
  2. Import the engine files (JS, CSS) 
    RPGcEngine.js
rpgcEngine.css
  3. Create a object to store a reference of the engine. 
    var RPG_c_Engine = {};
  4. Create a config object, this will store the setting the engine will use. 
    var config = {
    "scriptContent": scene1,
    "sceneCanvas": "#stage",
    "textSpeed": 24
};
    "scriptContent" is the script/transcript you want to use in the JSON format.
    "sceneCanvas" is the reference to the HTML element that display the scene.
    "textSpeed" adjust how fast the letter a typed into the speech bubbles.
  5. At this point the engine is set and ready to be used. You have passed in the script data it will use, informed where the stage (HTML Elem) is and pick the speed the text will be typed. To start the scene you will use the 'playScript' method within jQuery's onReady. 
    $(function(){
    RPG_c_Engine = RPGcEngine( config, config.sceneCanvas );
    RPG_c_Engine.playScript();
});
